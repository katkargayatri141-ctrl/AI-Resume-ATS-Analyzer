# 🤖 AI Resume ATS Analyzer

An AI-powered Resume ATS Analyzer that evaluates resumes, generates ATS scores, identifies strengths, weaknesses, missing skills, and provides personalized suggestions using Google Gemini AI.

---

## ✨ Features

- Upload Resume (PDF)
- AI-powered ATS Score
- Professional Resume Summary
- Resume Strengths Analysis
- Resume Weaknesses Detection
- Missing Skills Identification
- Personalized Improvement Suggestions
- Resume Analytics Dashboard
- Job Description Matching
- Responsive User Interface

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/katkargayatri141-ctrl/AI-Resume-ATS-Analyzer.git
```

### Backend

```bash
cd backend

pip install -r requirements.txt
```

Create a `.env` file

```
GEMINI_API_KEY=YOUR_API_KEY
```

Run the server

```bash
uvicorn app.main:app --reload
```

### Frontend

Open `frontend/index.html` using Live Server.
